# MALIYA-MD

WhatsApp Pair Code and QR Code Generator.

## Project Structure
- `index.js`: Main entry point (Express server)
- `pair.js`: Routing for Pair Code generation
- `qr.js`: Routing for QR Code generation
- `mega.js`: MEGA.nz upload logic
- `pair.html`: Frontend UI

## Setup
- Port: 5000
- Host: 0.0.0.0
- Workflow: "Start application" (npm start)
